# Application portfolio MVP scope

The MVP is a lightweight strategic application portfolio catalog, not a Software Asset Management, CMDB, ITSM, contract, licensing, or technical landscape repository. We deliberately exclude user counts, costs, licenses, contracts, renewals, support ownership, environments, integrations, tech stack, business capabilities, and business processes so the first version stays focused on cataloging Applications, related master data, ownership, lifecycle, TIME, PACE, fit, criticality, data-risk indicators, and catalog quality signals.

Future versions may add tech stack, business capabilities, business processes, integrations, CMDB environment data, contracts, licenses, costs, renewals, user counts, support ownership, diagnostic attachments or forms, verification workflows, and directory integration.
