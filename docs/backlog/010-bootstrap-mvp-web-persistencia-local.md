# 010 - Bootstrap Do MVP Web Com Persistencia Local

## Tipo

AFK

## Pai

[001 - Application Portfolio MVP PRD](./001-application-portfolio-mvp-prd.md)

## O Que Construir

Crie a base navegavel do MVP web de Application Portfolio, executando inteiramente no browser, com persistencia local e dados seed suficientes para avaliar o fluxo inicial sem backend.

## Criterios De Aceite

- [ ] O MVP abre localmente no navegador sem depender de backend.
- [ ] Os dados sao persistidos em armazenamento local do browser.
- [ ] O app inicializa com seed data para Vendors, Departments, Business Areas e Applications.
- [ ] A estrutura inicial permite navegar pelas areas principais do catalogo.

## Bloqueado Por

Nenhum - pode comecar imediatamente

## Verificacao

Abrir o MVP no navegador, confirmar que os dados seed aparecem, modificar um dado visivel, recarregar a pagina e confirmar que a alteracao persiste.
